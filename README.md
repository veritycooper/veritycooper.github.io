# Verity Cooper Portfolio

Static portfolio website designed for GitHub Pages.

## Files

- `index.html` contains the portfolio content.
- `styles.css` contains the visual design and responsive layout.

## Publish With GitHub Pages

1. Create a GitHub repository, for example `veritycooper.github.io`.
2. Upload `index.html` and `styles.css` to the repository.
3. In GitHub, open **Settings > Pages**.
4. Set the source to the main branch and root folder.
5. Save, then wait for GitHub Pages to publish the site.
